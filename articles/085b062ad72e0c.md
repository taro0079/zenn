# kakouneとは
kakouneとはTUIのテキストエディタ。
マルチカーソルに標準対応しているし、コマンドも直感的でわかりやすいので、個人的にはvimよりも優れているテキストエディタだと思っている。
kakouneでは検索機能などを標準で用意していない。外部コマンドを利用して実現することが求められている。
そのため、kakouneは外部コマンドと連携しやすい作りになっている。
今回は、https://strongly-typed-thoughts.net/blog/kakoune-philosophy を参考に`fd`と`git ls-files`を利用してファイルピッカーを実現してみる。


# ファイルピッカーの実現
```
# fd files picker
define-command my-file-picker %{
    prompt -shell-script-candidates 'fd --type file --hidden' open: %{ edit -existing %val{text} }
}

# git files picker
define-command my-git-files-picker %{
    prompt -shell-script-candidates 'git ls-files' open: %{ edit -existing %val{text} }
}
```

これをkakrcに書いた後、kakouneを再起動して
```
:my-file-picker
```
としてコマンドを実行することでファイルを検索することができる。
