## WebMCPとはなにか？
MCPとはLLMが利用できるツールの共通プロトコルであり、LLMはこのインターフェースに準拠したツール群を利用することができます。
WebMCPとはWeb操作を行うMCPで、これによってweb操作をLLMが行うことが出来ます。
従来はplaywright MCPを利用してweb操作を実現していましたが、この方法ではLLMがdomを分析する必要があり、webの挙動を理解するのに時間とトークンがかかってしまいます。
これを解決するのがweb mcpです。

## 実装方法
[Web MCP](https://webmcp.dev/) を参考にして実装していきます。
以下のようなディレクトリ構成でファイルを配置します。

```

┣ index.html
```  

```html
  <script>
    // 簡易DB（配列）検索
    function searchProductsInDatabase(query) {
      const products = [
        { id: 1, name: "MacBook Pro", price: 248000 },
        { id: 2, name: "Magic Mouse", price: 10800 },
        { id: 3, name: "Mechanical Keyboard", price: 15800 },
      ];
      const q = String(query ?? "").toLowerCase();
      return products.filter(p => p.name.toLowerCase().includes(q));
    }

    // 価格フォーマット
    function formatPrice(price) {
      return new Intl.NumberFormat('ja-JP', { style: 'currency', currency: 'JPY' }).format(price);
    }

    // 検索結果を表示
    function displayResults(products, query) {
      const container = document.getElementById('resultsContainer');
      const countEl = document.getElementById('resultsCount');

      if (products.length === 0) {
        container.innerHTML = `
          <div class="no-results">
            <div class="no-results-icon">😕</div>
            <p>「${query}」に一致する商品が見つかりませんでした</p>
          </div>
        `;
        countEl.textContent = '0件';
        return;
      }

      countEl.textContent = `${products.length}件`;

      const html = `
        <ul class="product-list">
          ${products.map(product => `
            <li class="product-item">
              <div class="product-info">
                <span class="product-id">${product.id}</span>
                <span class="product-name">${product.name}</span>
              </div>
              <span class="product-price">${formatPrice(product.price)}</span>
            </li>
          `).join('')}
        </ul>
      `;
      container.innerHTML = html;
    }

    // ローディング表示
    function showLoading() {
      const container = document.getElementById('resultsContainer');
      container.innerHTML = `
        <div class="loading">
          <div class="spinner"></div>
          <p>検索中...</p>
        </div>
      `;
    }

    // 検索実行
    function performSearch(query) {
      showLoading();

      // 少し遅延を入れてローディングを見せる
      setTimeout(() => {
        const results = searchProductsInDatabase(query);
        displayResults(results, query);
      }, 300);
    }

    // フォーム送信イベント
    document.getElementById('searchForm').addEventListener('submit', (e) => {
      e.preventDefault();
      const query = document.getElementById('searchQuery').value.trim();
      performSearch(query);
    });

    // 初期表示なし（WebMCPデモ用）

    // WebMCP 初期化 & ツール登録
    const mcp = new WebMCP();

    mcp.registerTool(
      "searchProducts",
      "商品名で商品を検索して一覧を返します（データのみ取得）",
      { query: { type: "string" } },
      (args) => {
        const products = searchProductsInDatabase(args.query);
        return {
          content: [{
            type: "text",
            text: JSON.stringify({ products }, null, 2)
          }]
        };
      }
    );

    mcp.registerTool(
      "performSearchOnUI",
      "検索フォームにキーワードを入力して検索ボタンをクリックし、画面に結果を表示します",
      { query: { type: "string", description: "検索キーワード" } },
      (args) => {
        const searchInput = document.getElementById('searchQuery');
        const query = args.query || '';

        // 入力フィールドにキーワードをセット
        searchInput.value = query;

        // 検索を実行
        performSearch(query);

        const products = searchProductsInDatabase(query);
        return {
          content: [{
            type: "text",
            text: `検索キーワード「${query}」で検索を実行し、画面に${products.length}件の結果を表示しました。`
          }]
        };
      }
    );
  </script>
```
